# 💼 InclusivaJobs

O InclusivaJobs é uma plataforma web desenvolvida com ASP.NET Core MVC voltada para inclusão e gerenciamento de oportunidades profissionais.

O projeto foi desenvolvido como atividade acadêmica da FIAP com foco em arquitetura web moderna, organização em camadas e persistência de dados.

---

## 🚀 Tecnologias Utilizadas

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap
- jQuery
- Razor Pages (CSHTML)
- Docker
- AutoMapper

---

## ✨ Funcionalidades

- Autenticação de usuários
- Cadastro de usuários
- Gerenciamento de vagas
- Listagem de oportunidades
- Persistência de dados com Entity Framework
- Interface responsiva com Bootstrap
- Arquitetura MVC organizada em camadas
- Paginação de dados

## ⚙️ Como Executar o Projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/seu-usuario/inclusivajobs.git
```

### 2️⃣ Acesse o diretório

```bash
cd inclusivajobs
```

### 3️⃣ Configure o banco de dados

Edite o arquivo:

```bash
appsettings.json
```

---

### 4️⃣ Execute as migrations

```bash
dotnet ef database update
```

---

### 5️⃣ Execute o projeto

```bash
dotnet run
```

A aplicação estará disponível em:

```bash
https://localhost:5001
```
